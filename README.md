# Open Source Software Lab — Shell Scripting Project

## Project Title

Shell Scripts for System Analysis and Automation (Python-Based Study)

## Author

Aanya Pahwa

## Description

This repository contains a collection of five shell scripts developed as part of an Open Source Software lab assignment.
The scripts are written using Bash and executed in a Linux-like environment (Git Bash).

The main objective of this project is to understand basic shell scripting concepts such as variables, conditional statements, loops, file handling, and system commands, while exploring the use of Python as an open-source software tool.


## Technologies Used

* Bash Shell Scripting
* Git Bash / Linux Terminal
* Python (for inspection and integration)

## Scripts Included

### 1. system_report.sh

Displays system-related information such as:

* OS name
* Kernel version
* Current user
* Uptime
* Date and time

### 2. package_inspector.sh

Checks Python installation and provides options to:

* View Python version
* Find Python installation path

### 3. directory_audit.sh

Performs directory analysis by displaying:

* Directory owner
* Permissions
* Size of directories

### 4. log_analyzer.sh

Analyzes a log file and:

* Counts occurrences of the word **"error"**
* Displays the last 3 matching lines

### 5. manifesto_generator.sh

Generates a simple open-source manifesto by:

* Taking user input
* Creating and writing to a file

## How to Run the Scripts

1. Open Git Bash or terminal
2. Navigate to the project folder:

   ```bash
   cd /c/Users/Aanya/OneDrive/Desktop/scripts
   ```
3. Give execution permission:

   ```bash
   chmod +x *.sh
   ```
4. Run any script:

   ```bash
   ./script_name.sh
   ```
